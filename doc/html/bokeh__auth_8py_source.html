190  There are issues storing hashed credentials and so this is not

243  """@brief Remove a user from the stored hashes.
244  If the username is not present then this method will return without an error.
245  @param username The login username.
246  @return True if the username/password was removed"""

255  """@brief Check the credentials are valid and stored in the hash file.
256  @param username The login username.
257  @param password The login password.
